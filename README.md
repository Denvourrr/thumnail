# Creative Thumbnail Design Project

## About This Project  

This project showcases my work in mobile graphic design and thumbnail creation.  
Each thumbnail is designed to be visually appealing, professionally balanced, and optimized for social media or video platforms.  

The focus is on clean layouts, modern typography, and strong visual composition — all created using mobile design tools.

- Pixellab  
- Canva Mobile  
- Lightroom Mobile  
- Snapseed / Remini

## Highlights  

- Designed entirely using mobile tools  
- Professional-level composition and color balance  
- Optimized for YouTube and social media thumbnails  
- Clean, attractive, and modern design style
